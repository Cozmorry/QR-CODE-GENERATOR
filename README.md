# QR Code Generator

This is a simple Python script that generates QR codes using the `qrcode` library. The QR code generated can store URLs, text, or any other form of data.

## Features
- Generate QR codes for URLs, text, or custom data
- Save QR codes as PNG images
- Customize QR code settings such as version, error correction, box size, and border

## Requirements
- Python 3.6 or later
- `qrcode` library for generating QR codes
- `Pillow` library (optional) for image manipulation
